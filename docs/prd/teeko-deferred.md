# Teeko — Deferred Features Checklist

> **Version:** 1.0
> **Date:** 2026-04-04
> **Status:** Draft
> **Purpose:** Tracks all features explicitly deferred from the MVP. To be prioritised for Phase 2 and beyond.

---

## How to Use This Document

Each item is marked with a status:
- `[ ]` — Not started
- `[x]` — Completed / shipped

---

## Phase 2 — High Priority (Post-MVP)

| # | Feature | Description | Status |
|---|---------|-------------|--------|
| 1 | **Teeko Wallet** | In-app balance riders can top up and use to pay for rides. Requires Bank Negara e-money compliance. | [ ] |
| 2 | **Tipping** | Riders tip drivers in-app after trip completion. Tips credited to driver in full — no commission deducted. | [ ] |
| 3 | **Fare breakdown in receipt** | Show base fare, distance, time, and surcharge breakdown. Currently receipt shows total only. | [ ] |
| 4 | **Toll / surcharge banner** | Display toll costs and surcharges on the ride type selection screen before booking. | [ ] |
| 5 | **Carpool / ride-sharing** | Shared rides to reduce fares further. | [ ] |
| 6 | **Corporate accounts (B2B)** | Company billing and employee ride management. | [ ] |
| 7 | **Loyalty / rewards programme** | Points, vouchers, and referral rewards for riders. | [ ] |
| 8 | **FPX / ATM wallet top-up** | Additional top-up methods for Teeko Wallet once launched. | [ ] |
| 9 | **Advanced analytics dashboard** | Detailed reporting for ops and finance teams (admin). | [ ] |

## Phase 2 — Lower Priority

| # | Feature | Description | Status |

## Out of Scope — All Phases

These features will not be built at any phase.

| Feature | Reason |
|---------|--------|
| Delivery / logistics (food, parcels) | Outside Teeko's product scope |
| Rental / hourly booking | Outside Teeko's product scope |
| Intercity travel | Outside Teeko's product scope |
| Driver vehicle financing | Outside Teeko's product scope |

## Open Questions (Deferred Items)

1. What is the exact no-show fee amount charged to riders (credited to driver)?
2. Does early cashout carry a transaction fee?
3. What are the confirmed vehicle makes/models accepted per ride category (Go, Comfort, XL, Premium)?
4. Is there a maximum number of vehicles a driver can register per account?
5. What is the appeal review turnaround time for suspended driver accounts?
